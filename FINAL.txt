                      ÉTAPES AUTOMATISÉES:
═══════════════════════════════════════════════════════════════════════════════

   Le script va automatiquement:

   1. ✓ Vérifier que Docker est installé
   2. ✓ Lancer PostgreSQL 16 en Docker
   3. ✓ Installer toutes les dépendances npm
   4. ✓ Configurer la base de données (migrations Prisma)
   5. ✓ Lancer le Backend Express sur le port 3001
   6. ✓ Lancer le Frontend Next.js sur le port 3000
   7. ✓ Ouvrir automatiquement le navigateur

═══════════════════════════════════════════════════════════════════════════════

   2. Votre navigateur ouvrira: http://localhost:3000
   3. L'application sera complètement fonctionnelle
   4. Les services tourneront en continu
