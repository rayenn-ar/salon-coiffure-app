             ├──► choisir coiffeuse (voir dispo en temps réel)
             ├──► choisir date/heure disponible
             ├──► sélectionner 1+ service(s)
             ├──► confirmation → création RDV (statut: EN_ATTENTE)

  CLIENTE ───► (UC-C2) Annuler un RDV
             ├──► sélectionner RDV depuis historique
             ├──► règle métier : annulation possible si RDV > 24h (configurable)
             └──► si < 24h avant ──► message d'erreur, annulation refusée

  CLIENTE ───► (UC-C3) Voir historique RDV (paginé)
             ├──► voir RDV passés et futurs
             └──► filtres par page (max 50 par page)

             ├──► note de 1 à 5 étoiles
             ├──► commentaire (optionnel)
             └──► avis en attente de modération par admin

  CLIENTE ───► (UC-C5) Gérer son profil
             └──► modifier nom, prénom, téléphone

  CLIENTE ───► (UC-C7) Activer l'authentification 2FA
             ├──► scanner QR code (Google Authenticator, Authy, etc.)
             └──► confirmer avec code TOTP

      1. Saisir email → {POST /auth/forgot-password}
      2. Recevoir lien par email (token 15 min, hashed en base)
    [PAGE /reset-password?token=...&uid=...]
      3. Vérifier token → {GET /auth/verify-reset-token}
      4. Saisir nouveau mot de passe → {POST /auth/reset-password}

  TOUT UTILISATEUR CONNECTÉ ───► Se déconnecter
      → {POST /auth/logout} → session Redis supprimée + refresh token révoqué

  TOUT UTILISATEUR CONNECTÉ ───► Renouveler token silencieusement
      → {POST /auth/refresh} → rotation du refresh token (sliding window)

  COIFFEUSE / ADMIN ───► Vérification MFA obligatoire
      1. Se connecter → token avec mfaVerified: false
      2. Si MFA non configuré → {POST /mfa/totp/setup} → QR code
         Confirmer avec code TOTP → {POST /mfa/totp/verify}
      3. Si MFA déjà configuré → {POST /mfa/totp/verify} au login
      4. Token mis à jour avec mfaVerified: true → accès aux routes pro/admin

  SCÉNARIO A — Première visite et réservation
  ─────────────────────────────────────────────
    1. / (accueil) → voir services et coiffeuses
    2. /services → choisir une prestation
    3. /coiffeuses → consulter profil + avis
    4. /inscription → créer un compte (nom, email, tél, mdp)
    5. /connexion → se connecter
    6. /reservation → choisir coiffeuse + date/heure + service
    7. RDV confirmé → visible dans /mon-espace

  SCÉNARIO B — Journée type coiffeuse
  ─────────────────────────────────────
    1. /connexion → login
    2. /mfa → saisir code TOTP (obligatoire)
    3. /espace-pro → voir agenda de la semaine
    4. Confirmer RDV en attente (statut EN_ATTENTE → CONFIRMÉ)
    5. Enregistrer walk-in (client sans compte) → RDV présentiel
    6. Fin de service → passer RDV en TERMINÉ
    7. Saisir produits consommés (matières) → stock mis à jour
    8. /espace-pro → onglet clientes → voir ses clientes

  SCÉNARIO C — Administration quotidienne
  ─────────────────────────────────────────
    1. /connexion → login admin
    2. /mfa → saisir code TOTP (obligatoire)
    3. /admin → tableau de bord CA / RDV du jour
    4. Onglet stock → vérifier alertes stock faible
    5. Onglet services → créer/modifier une prestation
    6. Onglet coiffeuses → ajouter une nouvelle coiffeuse
    7. Onglet dépenses → saisir une dépense du jour
    8. /parametres → ajuster délai d'annulation
